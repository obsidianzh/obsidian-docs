បើអ្នកទើបតែបើក Obsidian អ្នកនឹងឃើញឯកសារ ២ ។​ មួយនៅក្នុង pane ខាងឆ្វេងជាឯកសារអត្ថបទដែលអ្នកអាចកែប្រែដោយប្រើ Markdown ។ មួយនៅខាងស្តាំជា _Preview_ នៃឯកសារអត្ថបទដដែល បើអ្នកចង់ដកវាចេញជាឯកសារ HTML ឬ PDF ។ ចាំថា Preview អាចមានរូបរាងខុសបន្តិច ដូចជា links មិននៅក្នុងតង្កៀប ។ ដើម្បីទៅកាន់ link សូមចុចលើវានៅលើ Preview pane ។ អ្នកអាចទៅតាម links នៅក្នុង Editor pane ដែរដោយចុច `Ctrl/Cmd` នៅពេលអ្នកចុចលើ link ។បើអ្នកចង់ទៅកំណត់ត្រាដែលអ្នកនៅអំបាញ់មិញ ចុច `Ctrl/Cmd-alt-LeftArrow` 

ផ្លូវមួយទៀត, កំណត់ត្រាដែល link ទៅកំណត់ត្រាដែលអ្នកកំពុងសរសេរអាចឃើញនៅក្នុង **Backlinks Panel** នៅខាងស្តាំ (ស្វែងយល់បន្ងែមពី Backlinks [[Backlinks|នៅទីនេះ ។]])

ប្រសិនបើអ្នកវង្វេង ឬ ចង់រុករកបន្ថែមអ្វីដែល Obsidian អាចធ្វើបាន សូមចុច `Ctrl/Cmd-P` ដើម្បីប្រើ Command Palette ។ អ្នកអាចសរសេរ commands នៅទីនេះដូចជា "Navigate back" ហើយអ្នកនឹងឃើញបញ្ជីនៃអ្វីដែលអ្នកអាចធ្វើបាន ជាមួយនឹង hotkey shortcuts បើអ្នកមាន ។

តាមធម្មតា, Obsidian ជាឧបករណ៍មួយសម្រាប់ធ្វើការជាមួយឯកសារ Markdown មួយចំនួនដែលត្រូវបានរក្សាទុកនៅលើកុំព្យូទ័ររបស់អ្នក ។ បើអ្នកមិនស្គាល់ Markdown ទេ វាគ្រាន់តែជាអត្ថបទធម្មតាដែលមានទ្រង់ទ្រាយមួយចំនួនដែលធ្វើឱ្យវាមានឥទ្ធិពល (សម្រាប់បន្ថែមលើនោះ សូមមើល [[រៀបចំ syntax ទូទៅ]]) ។ ជាលទ្ធផល ប្រសិនបើអ្នកចង់ប្រើ Obsidian សម្រាប់រក្សាកំណត់ត្រា ឬ អ្វីក៏ដោយ វាដើរយ៉ាងអស្ចារ្យ ។

ឧទាហរណ៍ អ្នកចង់សរសេរកំណត់ត្រាអំពីក្រុមដែលអ្នកចូលចិត្ត ។ សូមសរសេរឈ្មោះនៃកំណត់ត្រាខ្ចប់ដោយតង្កៀប ២ ដូចជា `[[ក្រុមដែលខ្ញុំចូលចិត្ត]]`


អ្នកនឹងឃើញថាអក្សរ និង តង្កៀបក្លាយជាពណ៌ខៀវ ។ នេះមានន័យថាវាជា link ។ ចុច link នោះនៅក្នុង Preview (ឬ `Cmd-click` នៅក្នុង Editor), ហើយអ្នកនឹងឃើញកំណត់ត្រាទទេ ។ សរសេរអ្វីដែលអ្នកចង់នៅទីនោះ ហើយពេលសរសេរហើយ ត្រឡប់មកវិញដោយ `Cmd-alt-left` ឬ ចុចឈ្មោះនៃកំណត់ត្រានេះ ([[ការកត់ត្រាសាមញ្ញ]]) នៅក្នុង **Backlinks** នៅខាងស្តាំ ។ ស្វែងយល់បន្ថែមពី navigating links សូមមើល [[Internal links]] ។
